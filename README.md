# ATER-Yellow-Source
The source files of ATER Yellow and Blue cursor theme

# How to build?
1. Install [xcursorgen](https://www.x.org/archive/X11R7.7/doc/man/man1/xcursorgen.1.xhtml)
2. Run the make.sh inside the source directory
